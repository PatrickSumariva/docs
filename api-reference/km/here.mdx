**Slug:** `here` · **Modo de transporte:** caminhão (`truck`)

A integração com a **HERE Technologies** usa a Geocoding API para resolver o CEP em coordenada e o Routing v8 para calcular a distância rodoviária.

## Diferenciais

- É o único provedor com **modo caminhão (`truck`) nativo**, o que tende a refletir melhor as restrições e rotas reais de transporte de carga.
- Routing v8 estável e maduro.
- Tende a ser uma opção mais **barata** e com **bons resultados**.

## Quando priorizar

Coloque a HERE no topo da [ordem de prioridade](/api-reference/km/visao-geral#ordem-de-prioridade) quando a precisão da rota de caminhão for o mais importante para a sua operação.

<Info>
  A chave de API da HERE é mantida pela calcANTT. Para usar, ative o provedor em **Integrações → API & Marketplace**, com o add-on contratado e ativo (**Configurações → Plano**).
</Info>
