---

Se você prefere uma documentação de integração mais direta, com a especificação OpenAPI pura e um cliente para testar os endpoints no próprio navegador, use o **Swagger UI** da calcANTT.

No Swagger você encontra:

- Todos os endpoints da API pública num formato enxuto.
- Os schemas de requisição e resposta direto da especificação OpenAPI.
- Um cliente embutido para enviar requisições de teste.

  Especificação OpenAPI e testes rápidos dos endpoints (Clientes / API Key).
</Card>

<Note>
  Para autenticar no Swagger, use uma chave de API gerada em **Integrações → API & Marketplace**. Veja [Autenticação API REST](/api-reference/authentication).
</Note>
