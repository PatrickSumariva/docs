---

Antes de chamar `/calculadora/calcular`, você pode buscar as listas de referência de configurações de veículos e tipos de carga suportados. Estes endpoints retornam os valores válidos para `numEixos` e `tipoCargaId` que o endpoint de cálculo aceita. Ambos os endpoints requerem a permissão `CALCULAR_FRETE`.

---

## Obter configurações de eixos de veículos

**`GET /api/public/v1/calculadora/tipos-veiculo`**

```bash
curl https://your-domain/api/public/v1/calculadora/tipos-veiculo \
  -H "Authorization: Bearer calt_live_sua_chave_aqui"
```

## Obter tipos de carga

**`GET /api/public/v1/calculadora/tipos-carga`**

```bash
curl https://your-domain/api/public/v1/calculadora/tipos-carga \
  -H "Authorization: Bearer calt_live_sua_chave_aqui"
```

[
  { "id": 1, "codigo": "GRA", "nome": "Graneleiro", "ordemExibicao": 1, "ativo": true },

### Campos da resposta

<Tip>
  Faça cache dessas listas de referência na sua aplicação — elas mudam apenas quando a ANTT publica uma nova versão da tabela. Refaça a busca quando notar respostas inesperadas de `VALIDATION_ERROR` referenciando IDs desconhecidos.
</Tip>
