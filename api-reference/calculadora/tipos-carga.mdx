Lista os tipos de carga aceitos pela calculadora. Use o `id` da entrada escolhida no campo `tipoCargaId` das requisições de cálculo e criação de carga.

<Tip>
  Busque novamente esta lista quando notar respostas inesperadas de `VALIDATION_ERROR` referenciando IDs desconhecidos.
</Tip>
