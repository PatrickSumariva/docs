CalcANTT is a platform for calculating Brazil's minimum road freight rates in compliance with the official ANTT (Agência Nacional de Transportes Terrestres) table. Whether you use the interactive web calculator, import cargo batches from Excel, or integrate your ERP or TMS via REST API, CalcANTT ensures every freight rate you submit meets regulatory minimums.

## Get started in 4 steps

<Steps>
  <Step title="Create your account">
    Sign up at the CalcANTT platform and activate your company account. You can invite team members from **Settings → Users**.
  </Step>
  <Step title="Choose a plan">
    Select the plan that fits your volume needs. Each plan defines monthly calculation limits, API key quotas, and available features. See [Plans](/plans) for details.
  </Step>
  <Step title="Generate an API key">
    Go to **Integrations → API Key & Marketplace** to create your first API key. Assign the `CALCULAR_FRETE` permission to enable freight calculations via the API.
  </Step>
  <Step title="Make your first calculation">
    Send a `POST` request to `/api/public/v1/calculadora/calcular` with your vehicle type, cargo type, distance, and load weight. See the [Quick Start](/quickstart) for a working example.
  </Step>
</Steps>
