---
title: "Planos da calcANTT: cotas, funcionalidades e limites de uso"
description: "Os planos da calcANTT definem limites mensais para requisições de API, cálculos, chaves de API e vagas na equipe, além de controlar quais funcionalidades como webhooks e importações estão disponíveis."
---

A calcANTT utiliza um modelo de acesso baseado em planos. Seu plano determina quantas requisições de API e cálculos você pode fazer por mês, quantas chaves de API pode gerar, quantos membros de equipe pode adicionar e quais funcionalidades estão habilitadas na sua conta. Todos os limites são resetados mensalmente, e você pode expandir cotas individuais com add-ons sem precisar fazer upgrade do plano inteiro.

## Limites do plano

Cada plano define os seguintes limites numéricos. Um valor de `-1` significa que o limite é ilimitado.

| Limite | Campo | Descrição |
| --- | --- | --- |
| Requisições de API mensais | `limiteRequisicoesMes` | Total de requisições HTTP que suas chaves de API podem fazer na API pública por mês |
| Cálculos mensais | `limiteCalculosMes` | Número de cálculos de frete (`POST /api/public/v1/calculadora/calcular`) por mês |
| Chaves de API | `limiteApiKeys` | Número máximo de chaves de API ativas que sua conta pode ter simultaneamente |
| Cota de importação Excel | `limiteImportacaoCargas` | Número máximo de registros de carga importados via Excel por mês. `0` significa que a funcionalidade está desabilitada; `-1` significa ilimitado |
| Vagas na equipe | `limiteUsuarios` | Número máximo de contas de usuário que podem ser adicionadas à sua empresa |

<Info>
  Você pode verificar seu uso atual em relação a cada limite a qualquer momento. Acesse **Configurações → Faturamento** no painel, ou veja o cartão de resumo de uso na tela inicial do painel.
</Info>

## Funcionalidades habilitadas por plano

Além dos limites numéricos, cada plano habilita um conjunto específico de funcionalidades do produto (`funcionalidades`). Uma funcionalidade deve estar incluída no seu plano para que aquela área do produto seja acessível.

| Chave da funcionalidade | O que desbloqueia |
| --- | --- |
| `CALCULADORA` | A interface da calculadora de frete baseada no navegador |
| `API_PUBLICA` | Acesso à API REST pública — necessária para qualquer integração programática |
| `IMPORTACAO_CARGAS` | Importação em massa de cargas via upload de arquivo Excel |
| `LOGS_INTEGRACAO` | Logs completos de auditoria de integração, incluindo detalhes de requisição/resposta e latência por chamada |
| `RELATORIOS` | Telas de relatórios com resumos históricos de cálculos e análises de uso |
| `SUPORTE_PRIORITARIO` | Canal de suporte prioritário com SLAs de resposta mais rápidos |
| `WEBHOOKS` | Notificações webhook HTTPS entregues aos seus sistemas quando eventos de cálculo ocorrem |

<Note>
  O acesso à API (`API_PUBLICA`) e a calculadora web (`CALCULADORA`) são funcionalidades separadas. Um plano pode incluir uma sem a outra. Verifique a lista de funcionalidades do seu plano se não conseguir acessar uma área específica do produto.
</Note>

## Planos trial

Novas contas começam em um plano trial. Os planos trial têm duração limitada: o campo `duracaoTrialDias` no plano define quantos dias o trial dura. Seu painel mostra a data de expiração do trial e o número de dias restantes.

Quando o período trial termina, sua conta passa para um estado inativo até que você assine um plano pago. Cálculos e acesso à API são suspensos durante este período, mas seus dados históricos e configurações são preservados.

<Warning>
  Configure um plano pago antes do seu trial expirar para evitar interromper qualquer integração em produção que dependa da API da calcANTT.
</Warning>

## Expandindo limites com add-ons

Se você precisar de mais capacidade sem trocar de plano, pode adquirir add-ons para expandir cotas individuais. Os seguintes tipos de add-on estão disponíveis (sujeito à configuração `permiteContratarAddons` do seu plano):

| Tipo de add-on | O que aumenta |
| --- | --- |
| `REQUISICOES` | Cota mensal de requisições de API (`limiteRequisicoesMes`) |
| `API_KEY` | Número de chaves de API permitidas (`limiteApiKeys`) |
| `CALCULOS` | Cota mensal de cálculos (`limiteCalculosMes`) |
| `IMPORTACAO` | Cota mensal de importação Excel (`limiteImportacaoCargas`) |

Cada add-on tem um valor de `incremento` que especifica quanto o limite aumenta por unidade adquirida. Para solicitar um add-on, acesse **Configurações → Plano** e navegue pelos add-ons disponíveis para o seu plano.

<Tip>
  Se você consistentemente atinge seus limites mensais, considere fazer upgrade do plano em vez de acumular vários add-ons — um plano de nível superior geralmente oferece um custo por requisição menor além de funcionalidades adicionais como `LOGS_INTEGRACAO` ou `SUPORTE_PRIORITARIO`.
</Tip>
