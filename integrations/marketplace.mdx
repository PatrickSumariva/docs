O marketplace de integrações da calcANTT permite conectar plataformas logísticas, provedores de mapas e ERPs para enriquecer seus fluxos de frete. Ativar uma integração desbloqueia capacidades adicionais na calculadora e na gestão de cargas, sem exigir desenvolvimento sob medida.

## Categorias do marketplace

As integrações são agrupadas por `CategoriaIntegracao` para ajudar você a encontrar o que precisa:

<CardGroup cols={2}>
  <Card title="ERP" icon="building">
    Sistemas de gestão empresarial. Sincronize fretes, registros de carga e dados de custo diretamente com seu ERP.
  </Card>

  <Card title="TMS" icon="truck">
    Sistemas de gestão de transporte. Conecte seu TMS para disparar cálculos e receber eventos do ciclo de vida das cargas via webhooks.
  </Card>

  <Card title="MAPAS" icon="map">
    Provedores de mapas e rotas. Habilite o cálculo de distância por CEP em vez de informar um `distanciaKm` fixo.
  </Card>

  <Card title="OUTROS" icon="plug">
    Outras integrações que não se encaixam nas categorias acima.
  </Card>
</CardGroup>

## Provedores de mapas

Quando uma integração de mapas está ativa, a calculadora de frete aceita `cepOrigem` e `cepDestino`, CEPs brasileiros, e busca a distância rodoviária automaticamente. Sem uma integração de mapas, você deve informar `distanciaKm` diretamente em cada requisição de cálculo.

A calcANTT suporta os seguintes provedores de mapas:

| Slug | Provedor |
| --- | --- |
| `here` | HERE Technologies |
| `google` | Google Maps Platform |
| `tomtom` | TomTom |
| `mapbox` | Mapbox |

<Note>
  As chaves de API dos provedores de mapas são mantidas pela própria calcANTT, você não precisa fornecer credenciais. Para usar um provedor, é necessário **contratar o add-on correspondente** em **Configurações → Plano** e mantê-lo **ativo no Marketplace** (**Integrações → API & Marketplace**).
</Note>

<Info>
  Durante o período de trial, a integração com a **HERE** é fornecida gratuitamente para que você teste o cálculo de distância por CEP sem precisar contratar um add-on.
</Info>

## Integração Lincros

A integração Lincros conecta a calcANTT diretamente à plataforma logística Lincros. Ativá-la desbloqueia a sincronização estendida de cargas, permitindo que os registros fluam entre os dois sistemas sem entrada manual de dados.

Para configurar o Lincros, ative o card no marketplace e siga os passos de conexão no painel de configuração que abre.

## Ativando uma integração

<Steps>
  <Step title="Abra o marketplace">
    Navegue até **Integrações → API & Marketplace** e selecione a aba **Marketplace**.
  </Step>
  <Step title="Encontre o card de integração">
    Navegue ou pesquise pela integração desejada. Cards indisponíveis no seu plano atual aparecem com um ícone de cadeado, o campo `disponivel` da integração controla isso.
  </Step>
  <Step title="Clique em Ativar">
    Clique no botão **Ativar** no card. Para provedores de mapas (HERE, Google, TomTom, Mapbox), basta deixar a integração ativa, a calcANTT cuida das chaves. Outras integrações (como Lincros ou ERPs) podem abrir um painel de configuração para você informar credenciais ou parâmetros do seu lado.
  </Step>
  <Step title="Conclua a configuração">
    Preencha os campos obrigatórios no painel de configuração e salve. O card é atualizado para mostrar a integração como ativa.
  </Step>
</Steps>

<Info>
  Algumas integrações exigem um add-on do tipo `RECURSO` que não está incluído no seu plano base. Se o card estiver bloqueado, acesse **Configurações → Plano** para solicitar o add-on ou fazer upgrade do seu plano.
</Info>

## Verificando a disponibilidade de uma integração

O campo `disponivel` em cada objeto `IntegracaoMarketplace` indica se a integração está disponível para o nível do seu plano. Integrações com `disponivel` igual a `false` ficam visíveis no marketplace, mas só podem ser ativadas quando seu plano passar a incluí-las.
